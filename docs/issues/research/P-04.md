## Likely answer

**MEDIUM-HIGH confidence**: Gesco is the sponsor/client. "Healthcare Support" (or "HCS") is the Gesco product brand, not a separate external client. No California company named "Gesco" or "HCS" surfaces in public IME market scans, suggesting pre-launch or very small operator.

**Assumptions that would flip this:**
- An LOI, purchase order, or email trail naming an external end customer.
- A white-label or co-branded UI stripe that was intentionally scrubbed.
- A tenant-seed row representing an external org.

## Recommended approach to obtain a definitive answer

- Query `bizfileonline.sos.ca.gov/search/business` for "Gesco" and "Healthcare Support" variants.
- Check CA DIR QME directory for any Gesco-affiliated physicians.
- Ask Gesco principals: client roadmap, target buyer persona, any signed LOIs.
- Search git history for client-specific strings: `git log --all -p | grep -iE "acme|client|customer|tenant.*seed"`.

## Default assumption if unanswered

Gesco is the sponsor and product owner. "Healthcare Support" is the product brand. No external end-client to pursue.

## Risk if assumption is wrong

If there IS a contracted external client: IP ownership, deliverable scope, and BAA signatories all need immediate clarification. Also: Gesco's HIPAA status (covered entity vs business associate) depends on the answer.
