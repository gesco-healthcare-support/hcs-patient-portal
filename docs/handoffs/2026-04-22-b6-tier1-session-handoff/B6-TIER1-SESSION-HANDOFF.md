## 5. Non-negotiable guardrails

From `CLAUDE.md` and `~/.claude/rules/`:

- **Never `ng serve`, `yarn start`, or `ng build --watch`.** Angular 20 + Vite breaks ABP DI (`NullInjectorError: CORE_OPTIONS`). Use `ng build --configuration development` + `npx serve`.
- **Windows path <= ~200 chars.** Microsoft.Data.SqlClient.SNI.dll fails past 260. W:\ is already short -- this is why Adrian set it up.
- **Every AppService has `[RemoteService(IsEnabled = false)]`.** Otherwise ABP registers duplicate routes.
- **Riok.Mapperly only, NOT AutoMapper / not `ObjectMapper.Map<>`.** Extend `MapperBase<TSource,TDest>`, decorate with `[Mapper(RequiredMappingStrategy = RequiredMappingStrategy.Target)]`, register in `CaseEvaluationApplicationMappers.cs`.
- **Every concrete EF test class: `[Collection(CaseEvaluationTestConsts.CollectionDefinitionName)]`.** Every repository test body: `await WithUnitOfWorkAsync(async () => { ... })`.
- **Synthetic data only.** Bogus v35.6.5 (+ Bogus.Healthcare) with `Randomizer.Seed = new Random(20260420)`. PHI scanner hook will block real data.
- **Branch names describe work.** `test/applicant-attorneys-coverage`, not `feat/b6-tier1-pr1e`.
- **Commit format:** `test({entity-kebab}): add <scope> (B-6 T1)`. See `~/.claude/rules/commit-format.md`.
- **Admin merge only after all CI green:** `gh pr merge --admin --squash`.
